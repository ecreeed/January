
# Lost Cat - January

Lost Cat is a survival horror game where the player controls a cat searching for 4 keys to escape an abandoned factory. 

A invisible monster is chasing the cat through the factory and can be seen through the movement of the dust in the air. Don't let the beast catching you!


## Monthly Demo
